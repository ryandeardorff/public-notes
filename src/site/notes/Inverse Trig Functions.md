If $x=\sin\theta$,
	$\theta = \arcsin x$
	$(\frac{-\pi}{2}\le\theta\le \frac{\pi}{2})$
---
If $x = \cos\theta$,
	$\theta = \arccos x$
	$(0 \le \theta \le \pi)$
---
If $x=\tan\theta$,
	$\theta = \arctan{x}$
	$(-\frac{\pi}{2}\le \theta \le \frac{\pi}{2})$
