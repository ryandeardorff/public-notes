Given: $p\rightarrow q$

| Contrapositive: | $\neg q\rightarrow \neg p$ |
| ---- | ---- |
| Converse: | $q\rightarrow p \quad (p\leftarrow q)$ |
| Biconditional | $p\leftrightarrow q$ |
