Flip one coin. If $T$, win \$1, if $H$ lose \$2. Let $X=$ \$ won/lost.

$$
\begin{align*}
P(x=1) &= \frac{1}{2}\\
P(x=-2 ) &= \frac{1}{2}
\end{align*}
$$
Is this a fair game?

No, $E[x]=1\cdot P(x=1) + (-2)P(x=-2)=1\cdot \frac{1}{2}-2 \cdot \frac{1}{2} = \frac{-1}{2}$
